# Frontend - Employee Management System

Frontend application built using React JS for Employee Management System.

## Live Application

http://43.204.68.152

## Features

- User Login / Registration
- JWT Authentication
- Role Based UI (Admin / User)
- Add Employee
- Update Employee
- Delete Employee
- Search Employees
- Pagination
- Responsive Design
- Loading Indicators

## Tech Stack

- React JS
- Axios
- JavaScript
- HTML
- CSS
- Bootstrap
